# ONNX ML Scoring

> **Enterprise Plugin**: ML Scoring is available as a separate plugin. See the Surgewave.Ai repository for documentation.

The ML Scoring feature provides ONNX model loading and real-time inference on streaming data. It is delivered as an enterprise .swpkg plugin that integrates with the Surgewave broker via the `IBrokerPlugin` interface.

## Installation

```bash
surgewave plugin install Kuestenlogik.Surgewave.Ai-x.y.z.swpkg
```
