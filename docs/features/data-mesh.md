# Data Mesh

> **Enterprise Plugin**: Data Mesh (Catalog) is available as a separate plugin. See the Surgewave.Governance repository for documentation.

The Data Mesh feature provides data catalog, domain ownership, and data product management. It is delivered as an enterprise .swpkg plugin that integrates with the Surgewave broker via the `IBrokerPlugin` interface.

## Installation

```bash
surgewave plugin install Kuestenlogik.Surgewave.Governance-x.y.z.swpkg
```
