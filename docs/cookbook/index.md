# Cookbook

Task-oriented recipes for common Surgewave scenarios. Each recipe is self-contained and copy-paste ready.

| Recipe | What it covers |
|--------|---------------|
| [Produce & Consume](produce-consume.md) | Surgewave Native Client, Confluent.Kafka, headers, batching, error handling |
| [Schema Registry](schema-registry.md) | Register Avro/JSON/Protobuf, produce with validation, schema evolution |
| [Streams Topology](streams-topology.md) | Filter/map, aggregation, stream-table join, windowed aggregation |
| [Connect Pipeline](connect-pipeline.md) | Source → Transform → Sink, CSV, PostgreSQL, Pipeline Designer UI |
| [QueueView](queue-view.md) | AMQP client, ack/nack/reject, DLQ handling |
| [Monitoring](monitoring.md) | Prometheus/Grafana setup, key metrics, consumer lag alerting |
