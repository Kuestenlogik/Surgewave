Up and running in three commands

Boot the broker, open the Control UI, send your first message. No cluster bring-up, no ZooKeeper, no waiting.

  1. 1

    Start the broker

    # Listens on :9092 (Kafka wire) and :9093 (Native)
surgewave broker start
  2. 2

    Open the Control UI

    # Topic browser, consumer groups, schema registry, plugins
surgewave control --urls http://localhost:5050
  3. 3

    Produce a message

    # Any Kafka client works — here's the Surgewave CLI
surgewave produce --topic orders --value '{"id":1,"total":42.99}'

  4. You're live

    Single-node Surgewave broker on :9092 (Kafka wire) and :9093 (native), Control UI on :5050, your first message on orders. From here: .NET quickstart, cookbook recipes, or scale to a cluster.
