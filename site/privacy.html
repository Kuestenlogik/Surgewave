Privacy Policy

What data this website collects and what Surgewave itself does.

The short version

This website serves marketing pages and documentation. It does not set tracking cookies, does not run analytics, and does not transmit any personal data to third parties beyond what a browser normally sends to fetch a page.

Surgewave itself runs on your infrastructure: the broker, Control UI, and CLI live on your machines and connect only to the endpoints you configure. There is no opt-out telemetry channel back to Küstenlogik. See the note below.

What this website stores

  • LocalStorage only: your selected theme (theme = light / dark / auto) so your choice sticks across visits. That’s it. No tracking, no user identifiers, no advertising cookies.
  • No session cookies. No analytics SDK. No tag manager.

Third-party resources loaded by this site

  • Google Fonts — the typefaces used by this site are loaded from fonts.googleapis.com. When your browser fetches them, Google may log the request (IP, User-Agent) per their privacy policy.
  • shields.io and NuGet / GitHub APIs — live status badges (CI, version, stars) fetch from img.shields.io, api.github.com, and azuresearch-usnc.nuget.org. These requests originate from your browser, not from our server.

No other third parties are contacted.

Hosting

This site is hosted on GitHub Pages. The host records standard HTTP access logs (IP address, timestamp, requested path, User-Agent) for security and operational purposes. We do not access these logs for any analytics use.

Surgewave the broker

Surgewave — the broker, Control UI, and CLI you install via dotnet add package Kuestenlogik.Surgewave.Client, the MSI/deb/rpm installers, or the container images — runs entirely on your infrastructure:

  • No account required. No login or registration with Küstenlogik.
  • No opt-out telemetry. No phone-home. OpenTelemetry traces / Prometheus metrics go only to the OTLP endpoint you configure.
  • All topic data, segment logs, and tiered-storage payloads live on the disks and object stores you control.
  • Network traffic from the broker is exclusively to clients, peer brokers, schema registries, and storage backends you have configured.
  • Source is Apache 2.0 and public on GitHub — feel free to audit.

Your rights (GDPR)

Because this site collects virtually no personal data, there is very little to request. If you believe we do hold personal data about you (e.g., via the hoster’s access logs), you have the right to information, correction, deletion, and restriction of processing under Art. 15–18 GDPR. Contact us at info@kuestenlogik.com.

If you run Surgewave itself, all data handling happens on your infrastructure under your control.
